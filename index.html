Задание №1

Напишите функцию, которая возвращает промис, разрешающийся с каким-либо значением.

Задание №2

Создайте промис, который всегда отклоняется.

Задание №3

Напишите функцию, которая возвращает промис, который разрешается через 3 секунды с каким-либо значением.

Задание №4

Создайте две функции, каждая из которых возвращает промис. Одна из них должна разрешаться, а другая отклоняться. Затем напишите код, который обрабатывает оба промиса и выводит результаты их выполнения.

- Нажмите несколько раз

Задание №5

Создайте асихронную функцию, которая будет делать запрос к внешнему API (например, получать данные о погоде) и выводить результат в консоль.

Задание №6

Напишите асихронную функцию, которая будет читать данные из файла на сервере и возвращать из в виде строки.
